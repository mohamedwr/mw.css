Getting Started

First of all place <link> to use MW.CSS

<link rel="stylesheet" href="css/mw.css">

Then you have to use jquery because some components needs js plugins

<script src="js/jquery.js"></script>

Finally you have to use mw.js to make plugins work

<script src="js/mw.js"></script>
